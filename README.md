# Проект Skypro2

## Описание:

Проект Skypro2 создан в рамках изучение Python


## Тестирование:

В папке Tests находятся Unit тесты. Запустить можно с помощью команды pytest
В тестах используется используют параметризацию @pytest.mark.parametrize,
а также фикстуры из conftest.py. Чтобы добавить новую фикстуру используйте @pytest.fixture

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Marykhan23/skypro2.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование:

В проекте присутсвует 4 модуля
- generators - обработка транзакций и генерация номеров карт
- masks - форматирование номеров карт и счетов
- processing - фильтрация транзакций по дате и статусу
- widget - непонятно для чего to be continued
