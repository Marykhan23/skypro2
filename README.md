## Описание:

Проект Skypro2 создан в рамках изучение Python

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Marykhan23/skypro2.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
